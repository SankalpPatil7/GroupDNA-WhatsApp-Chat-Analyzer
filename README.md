


# GroupDNA – WhatsApp Chat Behavioral Analytics System

## Overview

GroupDNA is a Python-based analytics system that processes exported WhatsApp chat files and generates meaningful insights about group activity and participant behavior.

## Features

- Chat Parsing
- Group Overview
- Participant Analysis
- Activity Analysis
- Word Frequency Analysis
- NumPy Activity Heatmap
- Personality Detection
- Final Analytics Report

## Technologies Used

- Python
- NumPy
- datetime

## Concepts Used

- File Handling
- String Processing
- Lists
- Dictionaries
- Loops
- Functions
- NumPy Arrays
- Date & Time Handling
